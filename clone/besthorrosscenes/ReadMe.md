  - [x] 우측에는 사이트에 대한 컨텐츠가 존재 (제목, 동영상, 감독, 좋아요, 싫어요)
  - [x] small일 때는 1 column으로 가장 첫번째가 site header이고, 밑으로는 컨텐츠가 존재
- [ ] 컨텐츠는 (제목, 동영상, 감독, 좋아요, 싫어요)
